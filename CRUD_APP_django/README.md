# Hospital Management System

A Django CRUD application developed to practice backend development and database management.

## Features

- Create, update, delete and view records
- Django ORM
- Django Admin
- Form handling
- Template rendering
- SQLite database

## Tech Stack

- Python
- Django
- SQLite
- HTML
- CSS

## Run

```bash
pip install -r requirements.txt

python manage.py migrate

python manage.py runserver
```

## Learning Outcomes

- CRUD operations
- Django Models
- Django Views
- Django Templates
- Database relationships
