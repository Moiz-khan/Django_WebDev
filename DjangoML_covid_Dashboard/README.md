# COVID-19 Dashboard

A Django application for visualizing COVID-19 datasets and experimenting with machine learning workflows.

## Features

- CSV data processing
- Dashboard interface
- COVID-19 statistics
- Data visualization
- Django templates

## Tech Stack

- Python
- Django
- SQLite
- Pandas
- HTML
- CSS

## Run

```bash
pip install -r requirements.txt

python manage.py migrate

python manage.py runserver
```

## Learning Outcomes

- Data visualization
- Django dashboards
- Dataset processing
- Integrating data analysis into web applications
