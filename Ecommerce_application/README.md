# Django E-commerce Application

A multi-app Django project implementing the core functionality of an online shopping platform.

## Features

- Product management
- Shopping workflow
- Media handling
- Django Admin
- Modular Django apps

## Tech Stack

- Python
- Django
- SQLite
- HTML
- CSS
- JavaScript

## Run

```bash
pip install -r requirements.txt

python manage.py migrate

python manage.py runserver
```

## Learning Outcomes

- Django project architecture
- Multi-app development
- Backend design
- Product management
- File uploads
